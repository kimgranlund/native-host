Sortable

Column sort state cycling (asc → desc → none) with ARIA sync. Click a header to cycle sort direction. The consumer handles actual data reordering in the ui-sort event.

Basic Table Sort

<ui-controller> wraps a plain <div> and applies the sortable trait. Click column headers to cycle sort direction. ARIA aria-sort is synced automatically.

<ui-controller traits="sortable" sortable-selector="th">
  <div>
    <table class="sort-table">
      <thead>
        <tr>
          <th>Name</th>
          <th>Language</th>
          <th>Stars</th>
        </tr>
      </thead>
      <tbody>
        <tr><td>React</td><td>JavaScript</td><td>220k</td></tr>
        <!-- ... -->
      </tbody>
    </table>
  </div>
</ui-controller>

<script>
  const wrapper = document.getElementById('basic-wrapper');
  wrapper.addEventListener('ui-sort', (e) => {
    const { column, direction } = e.detail;
    const tbody = wrapper.querySelector('tbody');
    const rows = [...tbody.querySelectorAll('tr')];
    const headers = [...wrapper.querySelectorAll('th')];
    const colIndex = headers.findIndex(
      h => h.textContent.trim() === column
    );
    if (direction === 'none') return;
    rows.sort((a, b) => {
      const aVal = a.children[colIndex].textContent;
      const bVal = b.children[colIndex].textContent;
      const cmp = aVal.localeCompare(bVal, undefined, { numeric: true });
      return direction === 'asc' ? cmp : -cmp;
    });
    for (const row of rows) tbody.appendChild(row);
  });
</script>

Custom Selector

Using data-column attributes on headers with a [data-column] selector. Column identity comes from the attribute rather than text content.

<ui-controller traits="sortable" sortable-selector="[data-column]">
  <div>
    <table class="sort-table">
      <thead>
        <tr>
          <th data-column="city">City</th>
          <th data-column="country">Country</th>
          <th data-column="population">Population</th>
        </tr>
      </thead>
      <tbody>
        <tr><td>Tokyo</td><td>Japan</td><td>13.96M</td></tr>
        <!-- ... -->
      </tbody>
    </table>
  </div>
</ui-controller>

<script>
  const wrapper = document.getElementById('custom-wrapper');
  wrapper.addEventListener('ui-sort', (e) => {
    const { column, direction } = e.detail;
    const tbody = wrapper.querySelector('tbody');
    const rows = [...tbody.querySelectorAll('tr')];
    const headers = [...wrapper.querySelectorAll('[data-column]')];
    const colIndex = headers.findIndex(
      h => h.dataset.column === column
    );
    if (direction === 'none') return;
    rows.sort((a, b) => {
      const aVal = a.children[colIndex].textContent;
      const bVal = b.children[colIndex].textContent;
      const cmp = aVal.localeCompare(bVal, undefined, { numeric: true });
      return direction === 'asc' ? cmp : -cmp;
    });
    for (const row of rows) tbody.appendChild(row);
  });
</script>

API

Provider Usage

AttributeTypeDefaultDescription
sortable-selector string '' CSS selector for clickable header elements within the host.
sortable-disabled boolean false Disables sort cycling when true. Header clicks are ignored.

Events

EventDetailDescription
ui-sort { column, direction } Fired when sort state changes. column is the column name (from data-column or text content), direction is 'asc', 'desc', or 'none'. Consumer handles actual data reordering.

CSS

SelectorDescription
[aria-sort="ascending"] Set on the header element for the currently sorted column when direction is 'asc'.
[aria-sort="descending"] Set on the header element for the currently sorted column when direction is 'desc'.
