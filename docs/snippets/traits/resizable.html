Resizable

Composable trait for drag-to-resize functionality. Configure axis constraints, min/max sizes, and handle selectors. Dispatches ui-resize-start, ui-resize-move, and ui-resize-end events. Press Escape to cancel.

Horizontal Resize

Drag the right edge to resize horizontally.
Width: 300px
<ui-controller traits="resizable"
  resizable-handle-selector=".resize-handle-right"
  resizable-axis="horizontal"
  resizable-min="150"
  resizable-max="500">
  <div class="resize-panel" style="width: 300px;">
    Drag the right edge to resize horizontally.
    <div class="resize-handle resize-handle-right"></div>
  </div>
</ui-controller>

Both Axes (with min/max)

Drag the corner to resize in both directions. Clamped to 150-500px.
300 x 120
<ui-controller traits="resizable"
  resizable-handle-selector=".resize-handle-corner"
  resizable-axis="both"
  resizable-min="150"
  resizable-max="500">

  </div>
</ui-controller>

API

Provider Usage

AttributeTypeDefaultDescription
resizable-handle-selector string '' CSS selector for the resize handle element(s) within the host.
resizable-axis 'horizontal' | 'vertical' | 'both' 'horizontal' Constrains resize to one axis or both.
resizable-min number — Minimum size in pixels.
resizable-max number — Maximum size in pixels.
resizable-disabled boolean false Disables all resize behavior when true.

Events

EventDetailDescription
ui-resize-start { width, height } Fired on first pointer move after handle pointerdown.
ui-resize-move { width, height } Fired on every pointer move during resize.
ui-resize-end { width, height } Fired on pointer up.

Attributes (set during resize)

AttributeSet onDescription
[resizing] Host element Present while the element is being resized. Use for user-select: none.
