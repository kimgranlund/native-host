RangeSelectable

Composable trait for selecting a range of items. Supports two modes: drag (click-drag-release) and click (click start, hover to preview, click to commit). Sets [range-selected], [range-start], and [range-end] attributes.

Drag Mode — Calendar Grid (Provider)

<ui-controller> wraps a grid and applies the range-selectable trait. Click and drag across cells to select a range. Release to commit.

import { RangeSelectController } from '@nonoun/native-ui';

const grid = document.getElementById('click-grid');
const ctrl = new RangeSelectController(grid, {
  selector: '.cell',
  mode: 'click',
});

grid.addEventListener('ui-range-change', (e) => {
  const { startIndex, endIndex } = e.detail;
  log.textContent = `Preview: days ${startIndex + 1}–${endIndex + 1}`;
});

grid.addEventListener('ui-range-select', (e) => {
  const { startIndex, endIndex } = e.detail;
  log.textContent = `Committed: days ${startIndex + 1}–${endIndex + 1}`;
});

clearBtn.addEventListener('ui-press', () => {
  ctrl.clearRange();
});

RangeSelectController — Attach to Any Element

Use RangeSelectController to add range selection to an existing element without subclassing. Same events, same modes — no trait mixin required.

import { RangeSelectController } from '@nonoun/native-ui';

const list = document.getElementById('ctrl-list');
const ctrl = new RangeSelectController(list, {
  selector: '.list-item',
  mode: 'drag',
});

list.addEventListener('ui-range-select', (e) => {
  const { startIndex, endIndex, items } = e.detail;
  log.textContent = `Selected ${items.length} items (${startIndex}–${endIndex})`;
});

API

Provider Usage

AttributeTypeDefaultDescription
range-selectable-selector string — CSS selector for selectable items within the host. Required.
range-selectable-mode 'drag' | 'click' 'drag' drag: click-drag-release. click: click start, hover to preview, click to commit.
range-selectable-disabled boolean false Disables range selection when true.

Events

EventDetailDescription
ui-range-change { startIndex, endIndex, items } During drag/hover, fires as range changes. Use for live preview.
ui-range-select { startIndex, endIndex, items } Fired on release/commit. Use for final selection.

CSS Attributes

AttributeSet onDescription
[range-selected] Items within range Present on items within the selected range. Use for highlight styling.
[range-start] First item Present on the first item in the range. Use for start-cap border-radius.
[range-end] Last item Present on the last item in the range. Use for end-cap border-radius.
