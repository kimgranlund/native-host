FocusTrappable

Composable trait for trapping Tab focus within a container. When enabled, Tab and Shift+Tab cycle through focusable children without leaving the container. On enable, focus moves to [autofocus] or the first focusable element. On disable, focus returns to the previously focused element.

import { FocusTrapController } from '@nonoun/native-ui';

const region = document.getElementById('trap');
const trap = new FocusTrapController(region);

function show() {
  region.removeAttribute('hidden');
  trap.enable(); // focuses [autofocus] or first focusable
}

function hide() {
  trap.disable(); // restores focus to previous element
  region.setAttribute('hidden', '');
}

Without Autofocus

When no [autofocus] element exists, focus goes to the first focusable child.

API

Methods

MethodDescription
enable() Starts trapping focus. Moves focus to [autofocus] or first focusable child. Saves the previously focused element for restoration.
disable() Stops trapping focus. Returns focus to the previously focused element.
destroy() Disables and cleans up. Call in teardown().

Keyboard

KeyAction
Tab Moves focus to next focusable child. Wraps to first when at last.
Shift+Tab Moves focus to previous focusable child. Wraps to last when at first.
