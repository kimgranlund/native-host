Popoverable

Composable trait that extends Dismissable with CSS Anchor positioning. Provides wirePopover(anchor, popover) to set up anchor-name/position-anchor, and syncPopover(open) to show/hide with popover="manual".

Basic Popover

Uses PopoverController directly to wire an anchor button to a popover panel. Click outside or press Escape to close.

Toggle Popover

This popover is anchored to the button using CSS Anchor positioning. Click outside or press Escape to close.

import { PopoverController, signal } from '@nonoun/native-ui';

const host = document.getElementById('pop-host');
const popover = new PopoverController(host);
const open = signal(false);

popover.wirePopover(anchor, panel);

anchor.addEventListener('ui-press', () => {
  open.value = !open.value;
});

host.addEventListener('ui-dismiss', () => {
  open.value = false;
});

// In an effect:
popover.syncPopover(open.value);

Built-in: ui-select

ui-select uses Popoverable internally to anchor its dropdown listbox to the trigger button. CSS Anchor Positioning and dismiss-on-click-outside are handled automatically. 

<ui-select placeholder="Pick a framework"
  options='[
    {"value":"react","label":"React"},
    {"value":"vue","label":"Vue"},
    {"value":"svelte","label":"Svelte"}
  ]'></ui-select>

API

Methods

MethodDescription
wirePopover(anchor, popover) Sets up CSS anchor-name / position-anchor styles between the anchor and popover elements.
syncPopover(open) Call in an effect. Shows (showPopover()) or hides (hidePopover()) the popover based on the boolean open state. Also manages the dismiss layer.
destroy() Cleans up the dismiss layer and styles.

Events

EventDetailDescription
ui-dismiss — Inherited from Dismissable. Fired when click-outside or Escape occurs while the popover is the topmost layer.

CSS Pattern

SelectorDescription
[popover]:not(:popover-open) Must set display: none to prevent author-layer display from overriding UA hide.
position-area: block-end Anchors the popover below the trigger. Use position-try-fallbacks: flip-block for viewport clipping.
