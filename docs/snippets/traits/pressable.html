Pressable

Composable trait that handles pointer and keyboard press states. Sets a [pressed] attribute during press and dispatches ui-press on release. Supports pointer capture and Enter/Space keys.

Basic Press (Provider)

<ui-controller> wraps each element and applies the pressable trait. Click or press Enter/Space.

Click or press Enter
Another pressable
Disabled
 
<ui-controller traits="pressable">
  <div class="press-box" tabindex="0">
    Click or press Enter
  </div>
</ui-controller>

On ui-button (Built-in)

ui-button already uses the Pressable trait internally. The ui-press event and [pressed] attribute work out of the box.

Button Press Accent Press Disabled 
<ui-button>Button Press</ui-button>

<script>
  button.addEventListener('ui-press', (e) => {
    console.log('pressed!', e.detail.pointerType);
  });
</script>

Pointer Types

The ui-press event includes a pointerType in its detail — "mouse", "touch", "pen", or "keyboard".

Press me (check log)
 
<ui-controller traits="pressable">
  <div class="press-box" tabindex="0">
    Press me (check log)
  </div>
</ui-controller>

<script>
  box.addEventListener('ui-press', (e) => {
    console.log(e.detail.pointerType);
    // "mouse" | "touch" | "pen" | "keyboard"
  });
</script>

API

Provider Usage

AttributeTypeDefaultDescription
pressable-disabled boolean false Disables press behavior when "true". Pointer events are ignored.

Events

EventDetailDescription
ui-press { pointerType } Fired on release. pointerType is "mouse", "touch", "pen", or "keyboard".

CSS

SelectorDescription
[pressed] Present on the element while pointer/key is held down. Use for active-press styling.
