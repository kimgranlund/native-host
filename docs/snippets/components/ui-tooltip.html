<ui-tooltip>

Hover/focus-triggered informational popover with delay, anchor positioning, and fade animation.

Placements

Top (default) Tooltip above the button Bottom Tooltip below the button Left Tooltip to the left Right Tooltip to the right 
<ui-button>
  <span slot="label">Top (default)</span>
  <ui-tooltip placement="top">Tooltip above the button</ui-tooltip>
</ui-button>

<ui-button>
  <span slot="label">Bottom</span>
  <ui-tooltip placement="bottom">Tooltip below</ui-tooltip>
</ui-button>

On Different Elements

Primary Button Click to perform action More information available Hover text This is a plain text tooltip 
<ui-button variant="primary" intent="accent">
  <span slot="label">Primary Button</span>
  <ui-tooltip>Click to perform action</ui-tooltip>
</ui-button>

<span tabindex="0" style="cursor: help;">
  Hover text
  <ui-tooltip>This is a plain text tooltip</ui-tooltip>
</span>

Custom Delay

Instant (0ms) No delay! Default (500ms) Standard half-second delay Slow (1500ms) 1.5 second delay 
<ui-button>
  <span slot="label">Instant (0ms)</span>
  <ui-tooltip delay="0">No delay!</ui-tooltip>
</ui-button>

<ui-button>
  <span slot="label">Slow (1500ms)</span>
  <ui-tooltip delay="1500">1.5 second delay</ui-tooltip>
</ui-button>
