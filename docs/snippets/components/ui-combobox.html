<ui-combobox>

Coordinator element that wires a text input + filterable popover listbox into a combobox.

Basic Usage

Apple Banana Cherry Grape Lemon Mango Orange Peach Pear Strawberry 
<ui-combobox>
  <ui-input placeholder="Search fruits...">
    <ui-icon name="magnifying-glass" slot="leading"></ui-icon>
  </ui-input>
  <ui-listbox popover>
    <ui-option value="apple" label="Apple">Apple</ui-option>
    <ui-option value="banana" label="Banana">Banana</ui-option>
    <ui-option value="cherry" label="Cherry">Cherry</ui-option>
  </ui-listbox>
</ui-combobox>
